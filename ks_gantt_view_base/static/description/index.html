Gantt View Base

Odoo Gantt View Base is the basic framework that lays out the foundation required to implement Odoo Gantt View. It acts as the pre-cursor to unleash the powerful project management features of Gantt View to smoothly plan, manage, and accomplish projects under a fixed timeline.
